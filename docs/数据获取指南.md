### 选项3：HistData.com（手动下载）

**优点**：
- ✅ 完全免费
- ✅ 历史数据丰富（多年数据）
- ✅ 无需API密钥

**缺点**：
- ❌ 需要手动下载
- ❌ 数据格式需要转换
- ❌ 最小周期为M1（需要重采样）

**步骤**：

1. **访问网站**
   - https://www.histdata.com/download-free-forex-data/

2. **选择参数**
   - Currency Pair: `EUR/USD`
   - Time Frame: `M1` (1分钟)
   - Format: `ASCII` 或 `Generic ASCII`
   - Year/Month: 选择需要的时间范围

3. **下载并解压**
   - 下载ZIP文件
   - 解压得到CSV文件

4. **格式转换**
   ```python
   import pandas as pd
   
   # 读取HistData的M1数据
   df = pd.read_csv('DAT_ASCII_EURUSD_M1_202401.csv', 
                    names=['datetime', 'open', 'high', 'low', 'close', 'volume'],
                    parse_dates=['datetime'])
   
   # 重采样为M5
   df.set_index('datetime', inplace=True)
   df_m5 = df.resample('5T').agg({
       'open': 'first',
       'high': 'max',
       'low': 'min',
       'close': 'last',
       'volume': 'sum'
   }).dropna()
   
   # 保存
   df_m5.reset_index().to_csv('./backtest/data/EURUSD_5m_2024-2026.csv', index=False)
   ```

## 📊 数据格式要求

CSV文件必须包含以下列：

```csv
datetime,open,high,low,close,volume
2024-01-01 00:00:00,1.10000,1.10050,1.09950,1.10020,1000
2024-01-01 00:05:00,1.10020,1.10080,1.10010,1.10060,1200
2024-01-01 00:10:00,1.10060,1.10100,1.10040,1.10080,1500
...
```

**字段说明**：
- `datetime`: 时间戳（格式：YYYY-MM-DD HH:MM:SS）
- `open`: 开盘价
- `high`: 最高价
- `low`: 最低价
- `close`: 收盘价
- `volume`: 成交量（可以是Tick数量）

## ⚠️ 常见问题

### Q1: OANDA API限制
A: 免费账户有以下限制：
- 每次最多5000根K线
- 有请求频率限制（建议每次请求间隔1秒）
- 脚本已自动处理分批下载

### Q2: MT5连接失败
A: 检查：
- MT5终端是否正在运行
- 是否已登录账户
- 防火墙是否阻止连接

### Q3: 数据缺失
A: 外汇市场周末休市，数据中会有间隙，这是正常的。

### Q4: 需要多少数据？
A: 建议：
- **最少**: 3个月（约25,000根5分钟K线）
- **推荐**: 6-12个月（充分测试不同市场环境）
- **理想**: 2年以上
