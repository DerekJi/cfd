# 半自动趋势跟随交易系统

## 一、策略定位

| 项目 | 说明 |
|------|------|
| **策略类型** | 半自动趋势跟随 |
| **核心逻辑** | 人工判定大趋势 → 系统自动扫描信号 → 自动计算风控 → 人工确认开仓 → 程序执行下单 |

### 周期结构
- **H1周期**：识别趋势、标记趋势、绑定趋势均线
- **5M周期**：高频扫描、触发入场、执行开仓

---

## 二、核心模块

### 1. 品种观察池（Watchlist）
人工维护的待监控品种列表，系统定时扫描。

### 2. 趋势激活池（Trend Active Pool）
由人工标记多空方向、并绑定趋势均线的品种，进入高频监控。

---

## 三、全局规则

### 1. 数据抓取
| 周期 | 抓取范围 |
|------|----------|
| H1 | 最近200根K线 |
| 5M | 最近200根K线 |

### 2. 均线组（可通过 Telegram 命令修改）
| 周期 | 均线组 |
|------|--------|
| H1 | EMA20、EMA60、EMA80 |
| 5M | EMA20、EMA60、EMA80 |

### 3. 有效穿越定义（统一宽松规则）
- **多头**：价格进入均线上方 + 0.5×ATR 视为有效上穿
- **空头**：价格进入均线下方 - 0.5×ATR 视为有效下穿

---

## 四、第一步：品种观察池监控（H1周期）

| 项目 | 说明 |
|------|------|
| **监控对象** | Watchlist 内所有品种 |
| **执行频率** | 每小时执行一次 |

### 震荡区间规则
- 可为每个品种设置**震荡区间上限**和**下限**
- **价格在区间内**：不做任何提醒，不发送通知
- **价格突破区间**：恢复正常监控，触发均线穿越提醒

### 触发条件
价格在H1周期上，**突破震荡区间后**，有效穿越H1均线组中任意一条均线。

### 触发后动作
系统通过 Telegram 发送通知，包含：
- 最近200根H1 K线图表
- 叠加H1均线组（EMA20/60/80）
- 当前价格、均线数值、ATR
- 供人工选择：标记多头/空头/忽略

### 人工标记趋势
用户通过 Telegram 命令，为品种指定方向 + 绑定趋势均线：

```
/trend long EURUSD 20    # 标记多头，绑定 H1 EMA20
/trend short GBPJPY 60   # 标记空头，绑定 H1 EMA60
```

标记后，品种自动加入趋势激活池。

---

## 五、第二步：趋势激活池监控（5M高频扫描）

| 项目 | 说明 |
|------|------|
| **监控对象** | 仅趋势激活池内品种 |
| **执行频率** | 每5分钟执行一次 |

### 免打扰规则

#### 单品种临时免打扰
每次5M信号提醒时，提供免打扰选项：

| 命令 | 效果 |
|------|------|
| 15分钟免打扰 | 该品种15分钟内不再发送任何提醒 |
| 30分钟免打扰 | 该品种30分钟内不再发送任何提醒 |

- 免打扰期间，该品种信号仍正常检测，仅不推送通知
- 免打扰到期后自动恢复提醒

#### 全局免打扰时段
- 可设置**每日固定时段**全局免打扰
- 时段内所有品种（包括Watchlist和Trend Pool）均不发送提醒
- 支持多时段设置，如：
  - `00:00-06:00`（夜间）
  - `12:00-13:00`（午休）

### 顺势入场触发条件（满足任意一条即触发）
价格沿已标记的趋势方向，有效穿越以下任意一条均线：
1. 5M均线组中任意一条：EMA20 / EMA60 / EMA80
2. 该品种绑定的H1趋势均线

> **逻辑关系**：5M三条均线 + H1绑定均线 = 四条均线平等触发（或关系）

### 触发后附加过滤条件
| 方向 | 条件 |
|------|------|
| 多头 | 当前K线突破最近3根K线高点 |
| 空头 | 当前K线跌破最近3根K线低点 |

### 止损自动计算
| 方向 | 计算方式 |
|------|----------|
| 多头 | 最近10根K线低点下方 + 若干点（如 5 pip） |
| 空头 | 最近10根K线高点上方 + 若干点（如 5 pip） |

### 风控规则
- **单笔最大风险**：不超过账户资金0.8%
- **满足风控** → 推送开仓确认请求
- **不满足风控** → 仅推送信号，不提供开仓选项

---

## 六、第三步：开仓确认与执行

### 用户操作
| 命令 | 说明 |
|------|------|
| `/open` | 确认开仓 |
| `/cancel` | 取消信号 |

### 确认开仓后流程
1. 系统通过 OANDA 实时数据流（Stream API）获取最新价格
2. 在下一根5M K线范围内，在原入场价附近执行
3. 自动带入：入场价、止损位、计算好的仓位

### 取消信号
信号作废，等待下次触发。

## 七、Telegram 交互命令大全

### 1. 品种观察池管理
| 命令 | 说明 |
|------|------|
| `/watch add EURUSD GBPUSD` | 批量添加品种 |
| `/watch remove EURUSD` | 移除品种 |
| `/watch list` | 列出观察池 |

### 2. 震荡区间设置
| 命令 | 说明 |
|------|------|
| `/range set EURUSD 1.0850 1.0950` | 设置品种震荡区间（下限 上限） |
| `/range clear EURUSD` | 清除品种震荡区间 |
| `/range list` | 列出所有品种的震荡区间 |

### 3. 免打扰设置
| 命令 | 说明 |
|------|------|
| `/dnd 15` | 当前提醒品种15分钟免打扰 |
| `/dnd 30` | 当前提醒品种30分钟免打扰 |
| `/dnd global add 00:00 06:00` | 添加全局免打扰时段 |
| `/dnd global remove 00:00 06:00` | 移除全局免打扰时段 |
| `/dnd global list` | 列出所有全局免打扰时段 |
| `/dnd global clear` | 清除所有全局免打扰时段 |

### 4. 趋势激活池管理
| 命令 | 说明 |
|------|------|
| `/trend long EURUSD 20` | 标记多头 + 绑定 H1 EMA20 |
| `/trend short GBPJPY 60` | 标记空头 + 绑定 H1 EMA60 |
| `/trend clear EURUSD` | 清除趋势 |
| `/trend list` | 列出趋势池品种、方向、绑定均线 |

### 5. 参数配置
| 命令 | 说明 |
|------|------|
| `/set ema_h1 20 60 80` | 设置H1均线组 |
| `/set ema_5m 20 60 80` | 设置5M均线组 |

### 6. 开仓与订单
| 命令 | 说明 |
|------|------|
| `/open` | 确认开仓 |
| `/cancel` | 取消当前信号 |
| `/order list` | 查看持仓与订单 |

### 7. 持仓管理
| 命令 | 说明 |
|------|------|
| `/status` | 一键查看当前所有持仓的浮盈亏、止损位和距离 |
| `/close all` | 紧急情况（如重大新闻）一键全平仓 |
| `/move sl [品种] [价格]` | 快速手动将止损移动到保本位或新的支撑位 |

## 八、策略极简流程总结

```
1. 系统每小时扫 Watchlist 
   └── 检查是否在震荡区间内 → 区间内静默，区间外继续
   
2. 区间外 + H1穿越均线 → 发图给你
   
3. 你标记多空 + 绑定均线 → 加入趋势池
   
4. 系统5分钟扫一次：
   ├── 检查全局免打扰时段 → 时段内静默
   ├── 检查单品种免打扰 → 免打扰期内静默
   ├── 穿越5M均线组任意一条 或 穿越H1绑定均线 → 触发信号
   └── 信号推送附带免打扰选项（15分钟/30分钟）
   
5. 系统自动算止损、算仓位、过风控
   
6. 你点确认 → 程序用实时价格开仓
```
