## 全部修复清单 (12项)

### 第一阶段：核心风险管理 (Bug #1-5, Commit: fbeb5b9)
1. **复利bug** - 使用固定`total_account_size`参数替代动态`broker.get_value()`
2. **风险分配bug** - 总风险除以`num_symbols`，防止多品种风险叠加
3. **仓位计算公式** - 统一使用`size = risk_amount / (stop_distance × pnl_factor)`
4. **止损设置时机** - 通过`pending_entry_info`在订单完成后设置
5. **止损检查逻辑** - 检查`high[0]`/`low[0]`而非仅`close[0]`

### 第二阶段：滑点保护 (Bug #6-7, Commit: 40a65b3)
6. **滑点保护系数** - 仓位计算乘以3倍保守系数 (`SLIPPAGE_PROTECTION = 3.0`)
7. **最小手数风险验证** - 跳过因Round up导致过度风险的交易（1.5倍门槛）

### 第三阶段：ForexCommissionInfo (Bug #8-9)
8. **交叉货币对支持** - `ForexCommissionInfo`支持direct/indirect/cross三类货币对
9. **quote_usd_rate回归修复** - 非交叉货币对返回0.0而非1.0

### 第四阶段：仓位爆炸防护 (Bug #10-12) ⭐
10. **保证金上限** - 仓位保证金不超过可用资金的80% (`MAX_MARGIN_RATIO = 0.80`)
11. **最小止损距离** - 止损距离 ≥ ATR × 15%，拒绝极小止损交易 (`MIN_STOP_ATR_RATIO = 0.15`)
12. **单笔亏损仓位硬顶** ⭐ - 基于账户1%最大亏损和ATR估算，限制仓位上限

### 第五阶段：佣金修正 (Bug #13) ⭐ 新增
13. **佣金双重收取修正** - Backtrader开仓+平仓各调一次`_getcommission`，
    之前设`commission_per_lot=5.0`导致RT=$10（实际应为$5）。
    修正为`commission_per_lot=2.5`（$2.5×2=$5 RT），匹配Blue Guardian实盘数据

## Bug #10-12 根本原因分析

### USDJPY $83.65 / $58.91 亏损事件

**事件**（修复前）：2025-03-17，USDJPY SELL 61,365 units (0.61 lots)

| 项目 | 数值 |
|------|------|
| 入场价 | 148.698 |
| 止损距离 | 0.016 yen (0.011%) |
| 正常止损距离 | 0.5-2.0 yen |
| 实际退场价 | 148.826 |
| 实际移动距离 | 0.128 yen (8x 止损距离) |
| 原始亏损 | $52.78 |
| 佣金 | $6.14 |
| **净亏损** | **$58.91 > $50 FAIL** |

**根因**：
1. 策略不使用硬止损单，退场由EMA50趋势跟踪触发
2. 止损距离极小（0.016 yen）导致仓位极大（61,365 units）
3. 实际价格移动0.128 yen（8倍止损距离）后EMA50才触发退场
4. 3x滑点保护无法覆盖8x的实际偏离

**Fix 12 方案**：
```python
# 不依赖止损距离，直接基于ATR和最大亏损限额计算上限
MAX_SINGLE_LOSS_PCT = 0.01     # 账户1% = $50
ATR_WORST_CASE = 2.5           # 最坏退场距离 = 2.5×ATR (含佣金缓冲)
max_single_loss = total_account_size × MAX_SINGLE_LOSS_PCT
max_safe_size = max_single_loss / (ATR × ATR_WORST_CASE × pnl_factor)
size = min(size, max_safe_size)
```

**数学验证**（以问题交易为例）：
```
ATR ≈ 0.15 yen
pnl_factor = 1/148.698 = 0.006725
max_safe_size = 50 / (0.15 × 2.5 × 0.006725) = 19,826 units

修复前: 61,365 units → 亏损 $58.91 FAIL
修复后: 19,826 units → 亏损 ~$19.03 PASS ✅
```
