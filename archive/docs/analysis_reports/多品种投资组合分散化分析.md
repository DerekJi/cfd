## 五、为什么GBPUSD + USDJPY是最优组合？

### 5.1 数学原因

**投资组合理论**:
```
组合回撤 ≈ √(w1²σ1² + w2²σ2² + 2w1w2ρσ1σ2)

其中:
- w1, w2 = 权重 (各50%)
- σ1, σ2 = 波动率
- ρ = 相关系数 (0.076)

当ρ接近0时，组合波动率显著降低！
```

### 5.2 经济原因

**GBPUSD (稳定器)**:
- 成熟市场，流动性高
- 英国经济相对稳定
- 策略在此品种上胜率30.7% (最高)

**USDJPY (增长引擎)**:
- 日元避险属性，趋势明显
- 突破行情多，单笔盈利大
- 盈亏比4.91 (最高)

**组合效果**:
- GBPUSD在震荡市保护资本
- USDJPY在趋势市捕捉暴利
- 两者交易时机不重叠 → 回撤独立 → 组合回撤降低
